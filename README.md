# Антон Петров Борисовчи — 60 лет

Персональный сайт-биография к 60-летию. Создан как подарок от дочери.

## Стек

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## Разработка

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Сборка

```bash
npm run build
npm start
```

## Редактирование контента

Весь текстовый и медиа-контент задаётся в папке **`data/`** — менять компоненты не нужно.

- **`data/biography.ts`** — блоки биографии (заголовки, годы, текст).
- **`data/timeline.ts`** — события таймлайна (год, название, описание).
- **Секция «Проекты»** — карточки подставляются автоматически из папки **`public/images/projects/`**. Имя файла должно содержать название и год, например: `Название проекта (2020).jpg` или `Название - 2020.png`. После добавления картинок запустите `npm run projects:generate`, затем перезапустите dev или пересоберите проект.
- **`data/gallery.ts`** — фото галереи по категориям (детство, юность, семья, работа, путешествия). Картинки кладите в `public/images/` и укажите путь в `src`.
- **`data/stories.ts`** — истории от близких (автор, роль, текст).
- **`data/congratulations.ts`** — поздравления с юбилеем.
- **`data/lessons.ts`** — 60 уроков от папы.
- **Секция «Music»** — карточки альбомов подставляются из папки **`public/images/music/`**. Название карточки берётся из имени файла (без расширения), например: `Queen - Greatest Hits.jpg` → «Queen - Greatest Hits». После добавления обложек запустите `npm run music:generate`, затем пересоберите или перезапустите dev.
- **`data/family.ts`** — члены семьи (имя, отношение, описание, фото).
- **`data/letter.ts`** — финальное письмо (абзацы).

После правок в `data/*.ts` достаточно перезапустить или обновить страницу в браузере.

## Деплой на Vercel

1. Залейте репозиторий на GitHub.
2. Подключите репозиторий в [Vercel](https://vercel.com).
3. Сборка: `npm run build`, корневая директория — папка проекта.
4. Сайт будет доступен по ссылке, которую выдаст Vercel.
